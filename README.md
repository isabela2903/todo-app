# Todo App – React + Vite + Tailwind
Aplicação simples de lista de tarefas (Todo) feita com React, Vite e Tailwind CSS, permitindo adicionar, marcar como concluída, filtrar e remover tarefas.

## Funcionalidades
 * Adicionar novas tarefas.

 * Marcar tarefas como concluídas.

 * Filtrar por: All, Active e Completed.

 * Limpar tarefas concluídas.

 * Tema com estilos responsivos usando Tailwind CSS.

## Tecnologias utilizadas
 * React

 * Vite

 * TypeScript

 * Tailwind CSS

## Como rodar o projeto localmente
bash
```
# clonar o repositório
git clone https://github.com/isabela2903/todo-app.git

cd todo-app

# instalar dependências
npm install

# rodar em modo desenvolvimento
npm run dev
```

## Build para produção
bash
```
npm run build
npm run preview   # opcional, para testar o build localmente
```
Os arquivos otimizados ficarão na pasta dist/, prontos para deploy em serviços como GitHub Pages, Vercel ou Netlify.

## Scripts disponíveis
 * ```npm run dev``` – Inicia o servidor de desenvolvimento.

 * ```npm run build``` – Cria o build de produção.

 * ```npm run preview``` – Pré-visualiza o build de produção localmente.
